# Blueprint: Lotto Number Generator

## Overview

This project is a "Lotto Number Generator" application. It allows users to input their birthdate, birth time, and gender to generate personalized lottery numbers. The application is designed with a modern and clean user interface.

## Features

-   **User Input:** Users can enter their birthdate, birth time, and select their gender.
-   **Number Generation:** The application generates 5 sets of 6 lottery numbers based on the user's input.
-   **Fortune of the Day:** A "fortune of the day" is displayed along with the generated numbers.
-   **Save as Image:** Users can download the generated lottery numbers as a PNG image.
-   **Responsive Design:** The application is designed to be responsive and work on different screen sizes.

## Design

-   **Theme:** The application uses a dark theme with gold accents.
-   **Layout:** The layout is centered and easy to navigate.
-   **Styling:** The application uses modern CSS for styling, including CSS variables for easy theming.
